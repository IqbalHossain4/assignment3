<h2>Library Management System</h2>
Modern , scalable library management system.

<h3>Tech Stack</h3>
Node.js
Express.js
TypeScript
Mongoose
Database: MongoDB
<h3>How to get started</h3>
Clone the repository <br>
git clone https://github.com/IqbalHossain4/assignment3 <br>
<h4>Navigate to the project directory</h4>
cd assignment3 <br> 
<h4>Install dependencies</h4>
npm install <br>
Run the server <br>
npm run dev <br>
<h2>API</h2>
<h3>Book API</h3>
POST /api/books <br>
GET /api/books <br>
GET /api/books/:bookId <br>
PUT /api/books/:bookId <br>
DELETE /api/books/:bookId  <br>
<h3>Borrow Book API</h3> 
POST /api/borrow <br> 
GET /api/borrow <br>
